Detailed Description

template<typename TInput = uint16_t, typename TPrecision = half, Compute::DeviceType TDeviceType = Compute::DeviceType::Cuda>
requires ValidTensorTypes<TInput, TPrecision>
class Mila::Dnn::Encoder< TInput, TPrecision, TDeviceType >

An encoder module that provides token and positional embeddings.

The Encoder transforms input token IDs into continuous vector representations by:

  1. Looking up token embeddings from a vocabulary table (wte)
  2. Adding positional embeddings (wpe) based on sequence position

This implementation supports both CPU and CUDA execution depending on the template parameter.

Template Parameters
TInputThe input data type (typically int for token IDs)
TComputeThe computation data type (typically float)

Initialize the token and positional embedding tensors.

Creates and initializes the wte (word token embeddings) tensor of shape (vocab_len_, channels_) and the wpe (word position embeddings) tensor of shape (max_seq_len_, channels_). Both tensors are initialized using Xavier initialization.
