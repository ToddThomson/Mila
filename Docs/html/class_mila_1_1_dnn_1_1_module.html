Detailed Description

template<typename TInput, typename TCompute = TInput, Compute::DeviceType TDeviceType = Compute::DeviceType::Cuda>
requires ValidTensorTypes<TInput, TCompute>
class Mila::Dnn::Module< TInput, TCompute, TDeviceType >

Abstract base class for all modules in the Mila DNN framework.

Template Parameters
TInputData type of the input tensor elements.
TPrecisionData type of the compute tensor elements.
TDeviceDevice type, either CpuDevice or CudaDevice.
