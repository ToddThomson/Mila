Detailed Description

template<typename TInput, typename TPrecision = TInput, Compute::DeviceType TDeviceType = Compute::DeviceType::Cuda>
requires ValidTensorTypes<TInput, TPrecision>
class Mila::Dnn::Softmax< TInput, TPrecision, TDeviceType >

Softmax module for neural networks.

This class implements the softmax function, which is often used in the final layer of a neural network to convert raw scores into probabilities.

Template Parameters
TInputThe data type of the input tensor elements.
TPrecisionThe data type of the precision tensor elements.
TDeviceTypeThe device type, either Cpu or Cuda.
