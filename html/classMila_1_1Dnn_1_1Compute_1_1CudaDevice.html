Detailed Description

Class representing a CUDA compute device.

This class provides an interface to interact with NVIDIA CUDA-capable GPUs within the Mila framework. It handles device initialization, properties retrieval, and registration in the DeviceRegistry.

Registers all available CUDA devices with the DeviceRegistry.
