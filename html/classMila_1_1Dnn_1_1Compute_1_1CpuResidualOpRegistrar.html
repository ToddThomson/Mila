Detailed Description

Class responsible for registering the CpuResidualOp operation.

The CpuResidualOpRegistrar class registers the CpuResidualOp operation with the OperationRegistry. It associates the operation name "Cpu::ResidualOp" with a factory function that creates instances of CpuResidualOp.
