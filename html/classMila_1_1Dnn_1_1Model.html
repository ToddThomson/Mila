Detailed Description

template<typename TInput, typename TPrecision = TInput>
requires ValidTensorTypes<TInput, TPrecision>
class Mila::Dnn::Model< TInput, TPrecision >

A class representing a neural network model.

Template Parameters
TInputThe input data type for the model.
TPrecisionThe precision type used for model computations, defaults to input type.
