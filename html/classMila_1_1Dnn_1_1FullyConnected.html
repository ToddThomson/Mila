Detailed Description

template<typename TInput, typename TPrecision = TInput>
requires ValidTensorTypes<TInput, TPrecision>
class Mila::Dnn::FullyConnected< TInput, TPrecision >

A class representing a fully-connected module.

The fully-connected module (also known as a linear or dense layer) performs a linear transformation of the input data. The operation is defined as: output = input * weight + bias

This is a fundamental building block in neural networks that connects every input neuron to every output neuron with learnable weights.

Template Parameters
TInputThe data type of the input tensor elements.
TPrecisionThe data type used for computation and output (defaults to the input type).
