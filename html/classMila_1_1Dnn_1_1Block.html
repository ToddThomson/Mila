Detailed Description

template<typename TInput, typename TPrecision = TInput, DeviceType TDeviceType = DeviceType::Cuda>
requires ValidTensorTypes<TInput, TPrecision>
class Mila::Dnn::Block< TInput, TPrecision, TDeviceType >

A module class that can contain and manage child modules.

Block extends the base Module class with functionality to add, remove, and manage child modules. This is used for composite neural network components like MLPs, transformers, etc. that are built by composing simpler modules.

Template Parameters
TInputData type of the input tensor elements.
TPrecisionData type of the compute tensor elements, defaults to TInput.
TDeviceTypeThe device type for computation.
