Detailed Description

Class responsible for registering the CpuLayerNormOp operation.

The CpuLayerNormOpRegistrar class registers the CpuLayerNormOp operation with the OperationRegistry. It associates the operation name "Cpu::LayerNormOp" with a factory function that creates instances of CpuLayerNormOp.
