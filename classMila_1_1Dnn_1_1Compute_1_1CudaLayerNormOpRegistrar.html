Detailed Description

Class responsible for registering the CudaLayerNormOp operation.

The CudaLayerNormOpRegistrar class registers the CudaLayerNormOp operation with the OperationRegistry. It associates the operation name "Cuda::LayerNormOp" with a factory function that creates instances of CudaLayerNormOp.
